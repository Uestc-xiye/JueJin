# JueJin
 第四届字节跳动青训营大项目——仿掘金官网。

本项目实现了掘金官网的静态页面，包括导航栏、搜索框、会员、消息、登录、子导航栏，主体为无限文章列表和侧边栏。

还实现了签到、侧边栏广告、”回到顶部“按钮。
